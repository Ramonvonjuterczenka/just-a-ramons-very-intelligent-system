## 🚀 Quick Start

### Test Speech Recognition Logging
```
1. Open JARVIS in browser
2. Press F12 (Console)
3. Say "JARVIS"
4. See detailed logs with:
   - Language info
   - Confidence scores
   - Full transcripts
   - Wakeword detection
```

### Test Microphone Selection
```
1. Click SETTINGS
2. Find "Voice Activation Settings"
3. Click "🔄 REFRESH"
4. Select microphone from dropdown
5. Click "SAVE & REBOOT"
```
